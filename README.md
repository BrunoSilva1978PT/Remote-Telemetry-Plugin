# Remote Telemetry Plugin

A SimHub plugin that lets racing team members share live dashboards with each other. Every team member shares their own SimHub dashboard through a secure WebSocket tunnel. Teammates can view each other's dashboards remotely and inject raw telemetry data so their DDU, LEDs, and shift lights react to the remote driver's car.

## How It Works

1. Each team member runs SimHub with its built-in web server
2. The plugin creates a WebSocket tunnel through a relay server to expose each player's web server
3. Players create a team or join via single-use invite codes
4. Teammates view each other's dashboards via unique tunnel subdomains
5. Watch mode injects the remote driver's raw telemetry into your SimHub — dashboards, LED profiles, and DDU respond as if you were driving

## Features

### Teams & Roles
- **Create or Join Teams** — Create a team (you become the owner) or join via invite code
- **Multiple Teams** — Be part of several teams, each saved with its own token
- **Saved Teams** — One-click reconnect from the Connection tab
- **Roles** — Owner (full admin), Driver (share & watch), Engineer (spectator-only, browser-based)
- **Invite System** — Single-use invite codes with 48-hour expiry, generated by the owner
- **Transfer Ownership** — Owner can hand over the team to another member
- **Leave or Delete** — Drivers can leave; owners can delete the team entirely
- **Auto-Reconnect** — Automatic reconnection with exponential backoff

### Sharing
- **Automatic Tunnel** — Your dashboard is shared automatically when you connect
- **Shared Dashboard** — Choose which dashboard to display on the Race Engineer page
- **Sharing Toggle** — Start or stop sharing from the Session tab or mapped physical buttons
- **Viewer Count** — See how many people are watching your telemetry

### Watching Teammates
- **Watch Mode** — Click Watch next to a teammate to receive their live telemetry
- **Telemetry Injection** — Raw game data is injected into your SimHub via ReplayMode
- **Full DDU/LED Support** — Shift lights, flags, ABS/TC indicators all react to the remote car
- **Unavailable While Watching** — Players watching someone are hidden from others to prevent re-injected data chains
- **Auto-Resume** — Watching resumes automatically after a reconnect

### Race Engineer (Spectator Mode)
- **Browser-Based** — Web page at `/engineer` on the API server, no SimHub needed
- **Dashboard Grid** — View dashboards in 1, 2x1, 1x2, or 2x2 layouts
- **Multi-Monitor** — Launch separate windows on different monitors
- **Session Restore** — Automatically restore monitor windows from the last session
- **Auto-Connect** — Shareable URL with team and token for instant connection
- **Fullscreen Mode** — Toggle fullscreen for clean race-day setups
- **Dedicated Engineer Accounts** — Owner provisions engineer tokens in Team Admin

### Team Admin (Owner Only)
- **Generate Invites** — Single-use, 48-hour expiry, revocable
- **Add Engineers** — Create spectator accounts with dedicated tokens
- **Members Management** — View tokens, transfer ownership, kick members
- **Delete Team** — Permanently erase the team and disconnect everyone

### Custom Server Support
- **Saved Servers** — Add multiple custom API servers with a dropdown selector
- **Server Validation** — Servers are validated before being saved
- **Per-Server Token Storage** — Tokens are saved independently for each API server

### VPS Provisioning (Create Your Own Server)
- **One-Click Install** — Set up your own relay server on any VPS automatically via SSH
- **SSH Key Management** — Generate and manage SSH keys directly from the plugin
- **Server Status** — Real-time health check showing API and Relay status
- **DNS Records Helper** — Shows the required DNS records with resolved IPs
- **HTTPS Certificates** — Automatic certificate installation with ACME
- **Delete Server** — Uninstall all services from the VPS with one click

### Tunneling
- **Pure WebSocket Tunnel** — No external dependencies (no cloudflared/frpc needed)
- **HTTPS with Token Security** — Tunnel access requires a valid member token
- **Wildcard Subdomains** — Each player gets a unique subdomain (`name.tunnel.domain.com`)

### Other
- **Guide Tab** — Built-in documentation covering all features
- **About Tab** — Author info and donation link
- **Auto-Update Check** — Checks for new versions on GitHub Releases
- **Dark Theme** — Matches SimHub's native look with SimHub controls
- **Debug Logging** — Toggle debug logs from the Settings tab

## Requirements

- [SimHub](https://www.simhubdash.com/) with the built-in web server enabled (default port 8888)
- Windows 10/11
- Internet connection

## Installation

1. Download the latest `RemoteTelemetryPlugin.dll` from [Releases](https://github.com/BrunoSilva1978PT/Remote-Telemetry-Plugin/releases)
2. Close SimHub
3. Copy the DLL to your SimHub installation folder (e.g. `C:\Program Files (x86)\SimHub\`)
4. Open SimHub
5. The plugin appears in the left menu as **Remote Telemetry**
